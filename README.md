## Project Overview & Key Features
* Modern visual asset implementation utilizing high-end frosted glassmorphism style rules.
* Dynamic client-side theme engine allowing active switching across multiple UI color pallets.
* Fluid multi-device responsiveness utilizing relative flexbox and layout grid constraints.
* Enhanced interactive user experience containing elegant element hover states and custom transitions.
* Production-ready structural container architecture suited for modular portfolio integrations.

## Tech Stack
* **Markup:** HTML5
* **Styling & Effects:** CSS3 (Backdrop-Filters, Variable Shadows)
* **Interactivity Engine:** Vanilla JavaScript (ES6+)

## Getting Started

### Prerequisites
* A standard modern web browser with explicit parsing engine support for CSS backdrop filters.

### Installation
```bash
git clone [https://github.com/logusivam/Glassmorphism-Effect-Portfolio.git](https://github.com/logusivam/Glassmorphism-Effect-Portfolio.git)
```

### Usage
```bash
python -m http.server 8000
```
